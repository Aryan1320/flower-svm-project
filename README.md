# Flower Classification using HOG and SVM

## Overview
This project performs multi-class flower classification using:
- HOG (Histogram of Oriented Gradients)
- SVM (Support Vector Machine)

## Features
- Data augmentation
- Feature extraction using HOG
- Multi-class classification
- Top-3 predictions with confidence

## Dataset
Dataset is provided as a ZIP file and extracted during runtime.

## How to Run
1. Upload dataset.zip
2. Run all cells
3. Upload test image

## Output
- Displays uploaded image
- Shows Top-3 predictions with percentage
